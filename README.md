# vue-login-dom

## 一个简易的vue登录案例

## 用到的技术：
- 路由前置守卫
- 路由分配
- 共享数据仓库

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
